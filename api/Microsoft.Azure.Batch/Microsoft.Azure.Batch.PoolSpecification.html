    AutoScaleEnabled

    Gets or sets whether the pool size should automatically adjust over time.

    Declaration
    public Nullable<bool> AutoScaleEnabled { get; set; }
    Property Value
    Type Description
    Nullable<Boolean>
    Remarks

    If false, one of the TargetDedicatedComputeNodes or TargetLowPriorityComputeNodes property is required.

    If true, the AutoScaleFormula property is required. The pool automatically resizes according to the formula.

    The default value is false.

    AutoScaleEvaluationInterval

    Property Value
    Type Description
    Nullable<TimeSpan>
    Remarks

    The default value is 15 minutes. The minimum allowed value is 5 minutes.

    AutoScaleFormula

    Gets or sets a formula for the desired number of compute nodes in the pool.

    Declaration
    public string AutoScaleFormula { get; set; }
    Property Value
    Type Description
    String
    Remarks

    For how to write autoscale formulas, see https://azure.microsoft.com/documentation/articles/batch-automatic-scaling/. This property is required if AutoScaleEnabled is set to true. It must be null if AutoScaleEnabled is false.

    The formula is checked for validity before the pool is created. If the formula is not valid, an exception is thrown when you try to commit the PoolSpecification.
