    Gets the blob prefix to use when downloading blobs from an Azure Storage container. Only the blobs whose names begin with the specified prefix will be downloaded.

    Declaration
    public string BlobPrefix { get; }
    Property Value
    Type Description
    String
    Remarks

    This property is valid only when AutoStorageContainerName or StorageContainerUrl is used. This prefix can be a partial filename or a subdirectory. If a prefix is not specified, all the files in the container will be downloaded.

    FileMode

    Gets the file permission mode attribute in octal format.

    Declaration
    public string FileMode { get; }
    Property Value
    Type Description
    String
    Remarks

    This property is applicable only if the resource file is downloaded to a Linux node. This property will be ignored if it is specified for a ResourceFile which will be downloaded to a Windows node. If this property is not specified for a Linux node, then the default value is 0770.

    FilePath

    Gets the location on the compute node to which to download the file(s), relative to the task's working directory.

    Declaration
    public string FilePath { get; }
    Property Value
    Type Description
    String
    Remarks

    If the HttpUrl property is specified, this is required and describes the path which the file will be downloaded to, including the filename. Otherwise, if the AutoStorageContainerName or StorageContainerUrl property is specified, this is optional and is the directory to download the files to. In the case where this is used as a directory, any directory structure already associated with the input data will be retained in full and appended to the specified FilePath directory. The specified relative path cannot break out of the task's working directory (for example by using '..').

    HttpUrl

    Gets the URL of the file to download.

    Declaration
    public string HttpUrl { get; }
    Property Value
    Type Description
    String
    Remarks

    If the URL is Azure Blob Storage, it must be readable using anonymous access; that is, the Batch service does

    not present any credentials when downloading the blob. There are two ways to get such a URL for a blob in Azure

    storage: include a Shared Access Signature (SAS) granting read permissions on the blob, or set the ACL for the

    blob or its container to allow public access.

    StorageContainerUrl

    Gets the URL of the blob container within Azure Blob Storage.

    Declaration
    public string StorageContainerUrl { get; }
    Property Value
    Type Description
    String
    Remarks

    This URL must be readable and listable using anonymous access; that is, the Batch service does not present any

    credentials when downloading blobs from the container. There are two ways to get such a URL for a container in

    Azure storage: include a Shared Access Signature (SAS) granting read permissions on the container, or set the

    ACL for the container to allow public access.
