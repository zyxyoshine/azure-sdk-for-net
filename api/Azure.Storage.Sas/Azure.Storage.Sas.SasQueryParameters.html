    Class SasQueryParameters

    A SasQueryParameters object represents the components that make up an Azure Storage Shared Access Signature's query parameters. It includes components used by all Azure Storage resources (Containers, Blobs, Files, and Queues). You can construct a new instance using the service specific SAS builder types.
