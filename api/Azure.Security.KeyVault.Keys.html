    KeyClient

    The KeyClient provides synchronous and asynchronous methods to manage Key in the Azure Key Vault. The client supports creating, retrieving, updating, deleting, purging, backing up, restoring and listing the Key. The client also supports listing DeletedKey for a soft-delete enabled Azure Key Vault.
